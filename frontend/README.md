# Retina Stabilizer Frontend

React-based frontend for the Retina Video Stabilization system.

## Quick Start

### 1. Start Backend (Flask API)

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend runs on http://localhost:5000

### 2. Start Frontend (React)

```bash
cd frontend
npm install
npm start
```

Frontend runs on http://localhost:3000

## Features

- Video upload (drag & drop or file browser)
- Model selection (Neural or Classical)
- Real-time processing progress
- Side-by-side video comparison
- Comprehensive performance metrics
- Benchmark scores with stage breakdown

## Tech Stack

- React 18
- Tailwind CSS
- Axios
